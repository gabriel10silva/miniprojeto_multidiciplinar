O que é Front-End

O front-end é a parte de um site ou aplicativo que o usuário vê e com a qual interage. Tudo o que aparece na tela — textos, imagens, botões, menus e animações — faz parte do front-end. Ele é responsável por tornar a experiência do usuário visualmente agradável e funcional.

Linguagens Utilizadas:

HTML

O HTML (HyperText Markup Language) é a linguagem de marcação usada para criar a estrutura básica de páginas na web. Ele não é uma linguagem de programação, mas sim uma forma de organizar o conteúdo, dizendo ao navegador o que cada parte da página representa.

CSS

O CSS (Cascading Style Sheets, ou “Folhas de Estilo em Cascata”) é a linguagem usada para estilizar e deixar bonito o conteúdo de um site. Ele controla cores, fontes, tamanhos, espaçamentos, posições e até animações dos elementos criados com HTML.

JavaScript

O JavaScript é a linguagem que permite tornar um site interativo. Com ele, é possível fazer com que a página responda às ações do usuário, como clicar em botões, preencher formulários, abrir menus, mostrar alertas ou atualizar conteúdo sem precisar recarregar a página.
